Preguntas

  • ¿Qué diferencias y semejanzas hay entre Java y JavaScript?
    • Diferencias
      • 1. Java es un lenguaje de programación OOP, mientras que Java Script es un lenguaje de scripts OOP.
      • 2. Java crea aplicaciones que se ejecutan en una máquina o explorador virtual, mientras que el código JavaScript sólo se ejecuta en un explorador.
      • 3. El código Java necesita compilación, mientras que el código JavaScript está en todo el texto.
      • 4. Necesitan diferentes plugins.
    • Semejanzas
      • 1. Proceden del lenguaje C
      • 2. Tienen una sintaxis ligeramente parecida
  • ¿Qué métodos tiene el objeto Date? (Menciona al menos 5*)
    • 1. setFullYear()
    • 2. setMonth()
    • 3. setDate()
    • 4. setHours()
    • 5. setMinutes()
    • 6. setSeconds()
    • 7. setMilliseconds()
    • 8. setTime()
  • ¿Qué métodos tienen los arreglos? (Menciona al menos 5*)
    • 1. push() - Inserta un elemento al final del arreglo.
    • 2. unshift() - Inserta un elemento al inicio del arreglo.
    • 3. pop() - Remueve un elemento del final del arreglo.
    • 4. shift() - Remueve un elemento del principio del arreglo.)
    • 5. slice() - Crea una copia sombra del arreglo.
    • 6. Array.isArray() - Determina si el valor es un arreglo.
    • 7. length - Determina el tamaño del arreglo.
  • ¿Cómo se declara una variable con alcance local dentro de una función?
    • 1. Con la palabra clave var . Por ejemplo, var x = 42 . Esta sintaxis se puede utilizar para declarar variables locales y globales, dependiendo del contexto de ejecución.
    • 2. Con la palabra clave const o let . Por ejemplo, let y = 13.
  • ¿Qué implicaciones tiene utilizar variables globales dentro de funciones?

    • Las variables globales no declaradas a menudo pueden provocar un comportamiento inesperado. Por lo tanto, se desaconseja utilizar variables globales no declaradas.
